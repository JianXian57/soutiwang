# Codex 开发任务：自用刷题 APP v0.1 题库管理与手动搜索

## 项目目标

开发一个 Android APK，主要用于企业内部工人群体自建题库、导入题库、手动搜索题目并查看答案。

v0.1 只做本地题库管理，不做服务器、不做登录、不做云同步、不做 OCR、不做悬浮窗。

## 技术要求

- 平台：Android APK
- 推荐语言：Kotlin
- UI：Jetpack Compose，界面简洁，适合不太熟悉软件的用户使用
- 数据库：Room，本地保存
- Excel 读取：优先使用轻量 xlsx 读取库，例如 `fastexcel-reader`
- 文件选择：使用 Android 系统文件选择器，支持导入 `.xlsx`
- 数据只保存在本机

## 核心功能

### 1. 题库分类管理

题库需要支持分类/题库名称。

用户导入 xlsx 后：
- 自动根据文件名生成题库名称
- 进入导入编辑页
- 用户可以修改题库名称
- 用户确认后保存题库和题目

题库列表页面需要显示：
- 题库名称
- 题目数量
- 导入时间
- 可进入查看/搜索
- 可删除题库

### 2. xlsx 题库导入

导入表格常见列包括：

- 题干
- 答案
- 选项A
- 选项B
- 选项C
- 选项D
- 选项E
- 选项F

但表头不应强制完全一致，需要做自动识别。

#### 表头自动识别规则

第一行优先作为表头识别，但不要完全依赖固定列名。

需要尽量兼容以下写法：

题干列可能叫：
- 题干
- 题目
- 问题
- 试题
- 内容

答案列可能叫：
- 答案
- 正确答案
- 标准答案
- 参考答案

选项列可能叫：
- 选项A / A / A选项 / 答案A
- 选项B / B / B选项 / 答案B
- 以此类推到 F

如果无法识别表头，可以尝试按常见顺序推断：
- 第 1 列为题干
- 第 2 列为答案
- 后续列为选项 A-F

导入前需要显示预览页：
- 识别到的题库名称
- 总行数
- 可导入题目数
- 异常行数
- 前若干条题目预览
- 允许用户确认导入或取消

#### 导入防呆要求

导入时需要处理：

- 空行跳过
- 题干为空的行进入异常列表
- 答案为空的行进入异常列表
- 完全重复题干的题目提示重复，可默认跳过
- 选项为空时允许导入，不报错
- 表格中多余列忽略
- 单元格前后空格自动清理
- 换行、多个空格尽量归一化

异常列表需要在预览页展示简要原因，例如：
- 缺少题干
- 缺少答案
- 重复题目
- 无法识别列

### 3. 题型自动推断

不需要用户手动选择题型，系统根据答案和选项自动推断。

题型包括：
- 选择题
- 判断题
- 填空题
- 未知

建议规则：

选择题：
- 存在选项 A/B/C/D 等
- 或答案为 A、B、C、D、AB、ABC、A,B,C、选A 等形式

判断题：
- 答案为：正确、错误、对、错、是、否、√、×、true、false 等

填空题：
- 没有明显选项
- 答案不是判断题格式
- 答案为文本内容

无法判断时标记为“未知”。

### 4. 手动搜索

需要提供一个简单搜索页面。

用户输入关键词后，在当前题库或全部题库中搜索。

搜索范围：
- 题干
- 答案

搜索结果需要显示匹配性质：
- 匹配题干
- 匹配答案
- 题干和答案均匹配

搜索结果卡片显示：
- 题干
- 选项 A-F，空选项不显示
- 答案
- 题型
- 所属题库
- 匹配性质

支持点击按钮复制答案：
- 每个搜索结果提供“复制答案”按钮
- 点击后复制答案到剪贴板
- 显示 Toast：已复制答案

### 5. 页面结构建议

至少包含以下页面：

1. 首页 / 题库列表页
   - 显示所有题库
   - 按钮：导入题库
   - 按钮：搜索题目

2. 导入预览页
   - 显示识别结果
   - 可修改题库名称
   - 显示预览题目
   - 显示异常行
   - 按钮：确认导入
   - 按钮：取消

3. 搜索页
   - 搜索框
   - 题库范围选择：全部题库 / 某个题库
   - 搜索结果列表
   - 支持复制答案

4. 题库详情页
   - 显示该题库中的题目列表
   - 支持搜索该题库
   - 支持删除题库

## UI 风格

用户群体很多不太会使用软件，所以界面要简单直接：

- 首页突出“导入题库”和“搜索题目”
- 按钮文字明确，不使用复杂图标
- 导入失败或异常时，用普通中文解释
- 不要做复杂设置
- 不要做账号系统
- 不要做会员系统
- 不要做云同步

## 验收标准

完成后需要满足：

1. 可以安装运行 Android APK
2. 可以选择 `.xlsx` 文件导入
3. 可以自动识别题干、答案、选项 A-F
4. 可以在导入前预览题目和异常行
5. 可以修改题库名称并保存
6. 可以在本地保存多个题库
7. 可以搜索题干和答案
8. 搜索结果能标注匹配来源：题干/答案
9. 搜索结果能展示题干、选项、答案、题型
10. 可以一键复制答案
11. 没有服务器依赖，断网也能正常使用

## 暂不开发

v0.1 暂时不要开发以下功能：

- 悬浮窗
- OCR
- 截屏识别
- 自动检测屏幕
- 登录注册
- 云同步
- 付费/激活码
- 复杂相似度搜索
- AI 搜题
